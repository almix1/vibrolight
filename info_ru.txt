Программа vibrolite это попытка автоматизированной диагностики спектра вибрации.
С определенной долей вероятности программа определяет такие дефекты как: дисбаланс, расцентровка и дефекты подшипников качения (при наличии характеристик подшипника в файле 'bearing.txt').
Vibrolite тестовая программа и должна быть проверена при анализе реальных данных спектра экспортируемых в файл, в котором имеются два столбца, в первом частоты, во втором значения на частотах.
Спектры отображается с помощью программы gnuplot.
После старта программы, необходимо последовательно внести оборотную чаcтоту, ширину спектра и количество линий в спектре. Достоверность анализа будет зависит от того настолько точно будут соотвествовать внесенные данные тем которые содежатся в подключаемом файле.
В качестве примера программа может сама сгенирировать файл с данными для диагностики и провести диагностику на основании сгенирированных данных.
Для этого ее надо запустить с параметром -i: ./vib -i или ./vib -ip для отображения спектра в программе gnuplot.
Подключить файл для анализа можено с помощью: ./vib -f или ./vib -fp для отображения спектра в программе gnuplot. 
Программа использует файл 'bearing.txt' в котором содержится параметры подшипинков качения неоходимые для вычисления возможных дефектов подшипников качения. Файл содержит небольшое количество подшипков, при необходимости можете добавить в файл 'bearing.txt' необходимые подшипники и их характеристики.

Пример_1 запуска программы с генерацией спектра: ./vib -i
25 1000 2000
205
/* Результат: */
RMS: 6.05
disbalance: '4.45 - 25.00 Hz'
defect bearing: '3.75 - BPFO 90.00 Hz'
defect bearing: '1.56 - BPFI 135.00 Hz'
Программа создать файл с именем включающим дату, время и расширение csv.

Пример_1 запуска программы с данными из файла и отображением спектра в программе gnuplot:
./vib -fp 20240601121644.csv
25 1000 2000
205
/* Результат: */
RMS: 6.05
disbalance: '4.45 - 25.00 Hz'
defect bearing: '3.75 - BPFI 90.00 Hz'
defect bearing: '1.56 - BPFI 135.00 Hz'

Пример_2 запуска программы с генерацией спектра: ./vib -i
10 100 200
205
/* Результат: */
RMS: 5.75
disbalance: '4.04 - 10.00 Hz'
defect bearing: '3.12 - BPFO 36.00 Hz'
defect bearing: '2.55 - BPFI 53.50 Hz'
The program creates a file with a name that includes the date, time, and csv extension.

Пример_2 запуска программы с данными из файла и отображением спектра в программе gnuplot:
./vib -fp 20240601141354.csv
10 100 200
205
/* Результат: */
RMS: 5.75
disbalance: '4.04 - 10.00 Hz'
defect bearing: '3.12 - BPFO 36.00 Hz'
defect bearing: '2.55 - BPFI 53.50 Hz'

Соответственно:
25 или 10 - это оборотная частота, 1000 или 100 - ширина спектра, 
2000 или 200 - количество линий в спектре.
205 - номер подшипника.
